    Applicant Workflow

    1. Create or import an applicant record.
    2. Verify required documents.
    3. Move the application from Submitted to Under Review.
    4. Assign reviewer and document follow-up notes.
    5. Generate reports or export records for operational use.

    Testing Checklist

    • Verify required form fields.
    • Confirm status changes update dashboard counts.
    • Test filters, search, and export buttons.
    • Check incomplete application query accuracy.
    • Record issues and recommended improvements.

    Process Improvement Ideas

    • Automated reminders for missing documents.
    • Reviewer workload balancing.
    • Standardized ticket categories.
    • Monthly admissions trend reporting.
